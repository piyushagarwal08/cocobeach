= Includes =

* Real Estate Options: all settings need for a Real Estate website.
* Additional Custom Post Types, Taxonomies features: Property, Agent, Packages, User Packages, Invoices, Transaction Log
* With each Custom Post Types: Add New, Listing, Edit, Change, Delete and much more.

= Features =

* Already define pages for Real Estate website like: Search Page, Listing Properties, Listing Agent, Payment...etc
* Add new, edit, delete, listing all attributes of Property: Type, Status, Features, Cities, Neighborhood, Country/State, Label
* Add new, edit, delete, listing for Agent. Propably group more than 2 Agent become a Agency
* Have more type for payment: free, pay for package, pay per submission and uer can define features.
* Can define package for members
* Can buy package become a Agent or leave it to normal account
* Can management properties submited
* Submit Property from front end
* Boomark properties favourites
* Save Searched feature
* Filter display for Types, Citis, Status, County/State, Features
* Management invoices from front end

**For Real Estate Options(plugin options)**

* Can import/export a part of Options(Section) or entire Options
* Setting/Choose country, currency, language for date picker, measurement units, logo for print invoices
* Setting use min mode for js, css or not
* Define & setting up your pages like: my properties page, submit property form pages, advanced search page...etc
* Setting slug for your business page
* Setting price/money display format
* Setting allow compare properties or not
* Setting/define unlimited near place
* Setting/define/management email content which will send to customers for notify actions or completed
* Setting information display on invoices
* Setting somethings for google maps display
* Define submission form layout
* Setting for Advance Search, enable/disable Save Searched features
* Setting Social Profiles
* Settings Property, Agent pages: listing & single

**Payment Method Supported**

* Intergrated Paypal

**For Property**

* Add basic information with unlimited fields supported
* Add address & choose country location with google maps supported

**For Agent**

**For Packages**

* Define packages, include: free, have fee, time package, number of submission, price for 1 or all package, number of feature for each...etc
* Choose curent package display

**For User Packages**

* Search, listing all user with curent package applied.

**For Invoices**

* Management all invoices, can change/update status: Not Paid to Paid in Wire Transfer method

**For Transaction logs**

* Management all information of transaction on your website: time, payment method, buyer, money and status

== Installation ==

Do follow to install plugins:

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to menu Essential Real Estate -> Real Estate Options screen to configure the plugin
